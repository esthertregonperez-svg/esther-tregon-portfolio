
# 01-ConstructionDB

Base de datos MySQL para la gestión de una empresa de construcción.

Este proyecto consiste en el diseño e implementación de una base de datos relacional orientada a centralizar la información principal de una empresa de construcción: clientes, obras, presupuestos, empleados, materiales, stock, proveedores y facturación.

## Objetivo

Crear una base de datos estructurada, normalizada y segura que permita mejorar la organización de la información, reducir errores humanos y facilitar la toma de decisiones.

## Tecnologías

- MySQL
- MySQL Workbench
- SQL
- Modelo Entidad-Relación
- Vistas
- Triggers
- Roles y permisos
- Copias de seguridad

## Funcionalidades

- Gestión de clientes
- Gestión de obras
- Gestión de presupuestos
- Control de materiales y stock
- Gestión de proveedores
- Facturación
- Consultas SQL
- Vistas para informes
- Triggers para automatización
- Usuarios y permisos por rol

## Estado del proyecto

Proyecto académico finalizado.  
Próxima evolución: desarrollo de una aplicación web de flujo de trabajo conectada a la base de datos.
